Tema 01

Universo Gamer:
Interfaces de Alta Performance.

Os jogos digitais têm as interfaces mais avançadas do mundo. Sua missão é criar uma Landing Page ou Hotsite inspirada nesse universo.

O Cenário

O mercado de games exige excelência visual. Jogadores estão acostumados com interfaces futuristas, efeitos de neon, grids complexos e respostas instantâneas. Um site "quadrado" e sem vida não funciona aqui.

Este tema é o teste definitivo para unir Design de Interface (UI) com CSS Avançado.

A Sua Missão

Escolha um jogo que você gosta, um time de E-sports (real ou fictício) ou um torneio, e desenvolva uma página web imersiva. O site precisa ter:

  • Identidade Visual Forte: Cores, fontes e imagens que "gritam" o tema do jogo.
  • Layout Criativo: Fugir do padrão "bloco em cima de bloco". Use sobreposições.
  • Interatividade: Efeitos de hover (passar o mouse) impactantes nos cards e botões.

3 Caminhos Possíveis

Time de E-sports

Página de apresentação de uma organização (Loud, Pain, ou fictícia).

  • ✓ Cards dos Jogadores (Foto + Função)
  • ✓ Tabela de Próximas Partidas
  • ✓ Formulário "Seja um Patrocinador"

Wiki de Personagem

Página detalhada sobre um Agente (Valorant) ou Campeão (LoL).

  • ✓ Lore (História) com tipografia rica
  • ✓ Grid de Habilidades (Ícones + Descrição)
  • ✓ Galeria de Skins (Carrossel)

Página de Torneio

Divulgação de um campeonato escolar ou profissional.

  • ✓ Banner Hero com Data e Premiação
  • ✓ Tabela de Classificação (CSS Grid)
  • ✓ Botão de Inscrição (CTA Destaque)

Como executar?

1

Identidade Visual (Design)

Defina a "vibe". É terror? Use preto, vermelho e fontes rasgadas. É futurista? Use azul neon e fontes geométricas. Colete assets (imagens PNG sem fundo dos personagens) de alta qualidade.

2

Estrutura de Grid (Autoria Web)

Aqui o desafio é o layout. Use display: grid para criar os cards dos personagens. Eles precisam estar alinhados perfeitamente. Use background-image com gradientes sobrepostos para dar profundidade.

3

Efeitos Visuais (CSS + JS)

A cereja do bolo. Adicione hover:scale-105 nos cards. Use CSS Transitions para fazer os botões brilharem quando passar o mouse. Se possível, use JS para filtrar personagens por classe (Ex: mostrar apenas "Duelistas").
