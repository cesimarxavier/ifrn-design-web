Tema 03

Hotsite de Evento:
O Hype transformado em Código.

Eventos precisam de impacto imediato. Sua missão é criar uma página de alta conversão, com contagem regressiva e visual inesquecível.

O Cenário

Um "Hotsite" é um site com vida curta e objetivo único: fazer a pessoa clicar no botão de inscrição. Pode ser um Festival de Música, uma Feira de Tecnologia, um Hackathon ou um Campeonato de Skate.

Neste tema, o design precisa gerar urgência (O tempo está acabando!) e o código precisa ser robusto (Formulários que funcionam). É o equilíbrio perfeito entre marketing visual e lógica de programação.

A Sua Missão

Desenvolva uma página promocional para um evento fictício (ou real). O site não pode ser "morno". Ele precisa ter:

  • Tipografia Display: Títulos gigantes e expressivos.
  • Contagem Regressiva: Um timer real feito em JavaScript (Dias : Horas : Minutos).
  • Call to Action (CTA): Botões de destaque que chamam para a ação ("Comprar Ingresso").

3 Caminhos Possíveis

Festival de Música

Rock in Rio, Lollapalooza ou um festival indie local.

  • ✓ Line-up (Grade de atrações com fotos)
  • ✓ Cores vibrantes e gradientes
  • ✓ Efeito Parallax no scroll

Conferência Tech

Um evento tipo "Google I/O" ou um Hackathon do IFRN.

  • ✓ Cronograma (Timeline do evento)
  • ✓ Cards dos Palestrantes (Speakers)
  • ✓ Formulário de Inscrição validado

Première / Lançamento

Página de contagem regressiva para um filme ou jogo.

  • ✓ Trailer em destaque (Embed de Vídeo)
  • ✓ Fundo imersivo (Imagem Fullscreen)
  • ✓ Timer Gigante no centro

Como executar?

1

Impacto Visual (Design)

Eventos vendem emoção. Escolha uma fonte "Display" (grossa, estilizada) para os títulos. Use cores quentes (vermelho, laranja, amarelo) sobre fundo escuro para criar contraste e energia.

2

Seções Claras (Autoria Web)

A estrutura é vital. Crie seções semânticas: <section id="lineup">, <section id="tickets">. Use CSS Grid para alinhar as fotos dos artistas ou palestrantes de forma organizada e responsiva.

3

A Lógica do Tempo (JavaScript)

Aqui brilha o JS. Vocês precisarão usar setInterval para criar a contagem regressiva que atualiza a cada segundo. Também devem validar o formulário de inscrição (não deixar enviar e-mail vazio).

Dica de Ouro:

O botão de "Inscreva-se" (CTA) é o rei. Ele deve estar visível o tempo todo. Que tal deixá-lo fixo no topo (Sticky Header) ou fazê-lo pulsar levemente com CSS Animation?
