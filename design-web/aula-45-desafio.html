Contexto: Sites reais não são estáticos. Como freelancer, você precisa somar serviços rapidamente para passar um orçamento ao cliente.

Missão: Criar uma aplicação onde o usuário insere serviços e valores, e o sistema renderiza a lista e calcula o Total automaticamente em tempo real.
  • Defina o array servicos: [] no x-data.
  • Crie inputs para Nome e Valor (use x-model.number).
  • O botão "Adicionar" deve inserir um objeto no array.
  • Use x-for para exibir a lista visualmente.
  • Desafio Lógico: Crie um get total() que soma todos os valores e exibe o resultado no rodapé.
