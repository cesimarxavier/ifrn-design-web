Propriedade Position

Por padrão, o HTML segue um fluxo estático (um bloco após o outro). A propriedade position nos permite quebrar essa regra para sobrepor elementos, fixar barras de navegação e criar layouts complexos.

1

static

O comportamento padrão de todo elemento. Eles seguem o Fluxo Normal do documento.

Regra: As propriedades top, right, bottom, left e z-index NÃO funcionam aqui.

2

relative

O elemento permanece no fluxo normal, mas pode ser deslocado visualmente em relação à sua posição original.

  • O espaço original dele é preservado (como um fantasma).
  • Outros elementos não se movem para ocupar o lugar.
  • Serve de âncora para filhos absolutos.

top: 20px; left: 20px;

absolute

O elemento é removido do fluxo. Ele não ocupa espaço físico na página (flutua por cima).

A Regra de Ouro:
Ele se posiciona em relação ao ancestral posicionado mais próximo (geralmente um pai relative). Se não houver nenhum, ele se posiciona em relação à janela (body).

Pai Relative + Filho Absolute

div.pai { position: relative }

top: 0; right: 0;
bottom: 1rem;
4

fixed

O elemento é removido do fluxo e posicionado em relação à Viewport (Janela do Navegador).

Ele não se move quando a página é rolada (scroll). É usado para barras de navegação fixas, modais e botões de "Voltar ao topo".

sticky

Mistura de relative e fixed. O elemento age normalmente até atingir uma posição de rolagem definida (ex: top: 0), onde então se "cola" na tela.

Requisito: Só funciona enquanto o seu elemento pai estiver visível na tela.

Gerenciando Camadas (Z-Index)

Quando elementos são retirados do fluxo (absolute/fixed/relative), eles podem se sobrepor. O z-index define quem fica por cima.

z-index: 10
z-index: 20
z-index: 30

Quanto maior o número, mais "perto" do usuário o elemento está.
(Nota: z-index só funciona em elementos com position diferente de static)

Tabela de Decisão Rápida

Valor Sai do fluxo? Referência Uso Comum
static Não - Texto corrido, blocos padrão
relative Não (Só visual) Ele mesmo Ajustes finos, Pai de Absolute
absolute Sim Pai posicionado Ícones sobre imagem, Badges, Dropdowns
fixed Sim Viewport (Janela) Menu Fixo, Modal, Chat lateral
sticky Híbrido Scroll / Pai Cabeçalhos de tabela, Filtros laterais
