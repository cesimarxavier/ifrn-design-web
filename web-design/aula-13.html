A Propriedade Mágica: Flex

Na prática, raramente escrevemos grow, shrink e basis separados. Usamos o atalho flex. É a propriedade mais usada em layouts responsivos. 

/* O atalho comum */
.item {
    flex: 1; /* Cresce para ocupar tudo */
}

/* O que ele significa: */
.item {
    flex-grow: 1;
    flex-shrink: 1;
    flex-basis: 0%;
  }

Exemplo Clássico: Sidebar + Conteúdo

width: 100px
(Fixo)
flex: 1 (Ocupa o resto)

O conteúdo (roxo) cresce automaticamente para preencher todo o espaço que a sidebar (cinza) não usou.

2

Quebrando Regras com Align-Self

Imagine um gráfico de barras onde todas as barras começam de baixo (flex-end), mas o título do gráfico precisa ficar no topo. O align-self resolve isso.

Dica Pro: Use margin-left: auto em um container Flex para empurrar um único item para a extrema direita. É um truque antigo, mas funciona igual ao align-self no eixo principal!

O Botão no Rodapé do Card
