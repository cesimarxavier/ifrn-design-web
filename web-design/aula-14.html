Padrão 01

O "Media Object" (Imagem + Texto)

O layout mais comum da web. Usado em comentários, listas de produtos, tweets e cards de perfil. Uma imagem fixa na esquerda e texto flexível na direita.

.card { display: flex; gap: 16px; align-items: center; } /* A imagem tem tamanho fixo, o texto cresce */

Padrão 02

A Barra de Navegação

O segredo é o justify-content: space-between. Ele joga o primeiro filho para a extrema esquerda e o último para a extrema direita automaticamente.

.navbar { display: flex; justify-content: space-between; align-items: center; padding: 1rem; }
Padrão 03

O Rodapé Fixo (Sticky Footer)

Sabe quando a página tem pouco conteúdo e o rodapé sobe para o meio da tela? Isso é feio. Com Flexbox Column e flex-grow no conteúdo, o rodapé sempre fica embaixo.

body { min-height: 100vh; display: flex; flex-direction: column; } main { flex: 1; /* Empurra o footer */ }
