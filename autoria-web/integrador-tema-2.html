Tema 02

Agência Digital:
Transforme seu grupo em marca.

No mercado de tecnologia, o portfólio vale mais que o currículo. Sua missão é criar a identidade online do seu Squad.

O Cenário

Vocês não são apenas estudantes; são profissionais em formação. Designers, Desenvolvedores Front-end, Redatores. Mas quem sabe disso? Ninguém, a menos que vocês mostrem.

Neste tema, vocês simularão o lançamento de uma Agência Digital ou de um Estúdio Criativo. É a oportunidade perfeita para aplicar CSS Grid (para galerias) e Branding (criação de marca).

A Sua Missão

Crie um site institucional que venda os serviços do grupo. O site deve transmitir profissionalismo, criatividade e domínio técnico. Ele precisa conter:

  • Identidade Única: Nome da agência, logo e paleta de cores original.
  • Vitrine de Serviços: O que vocês fazem? (Web Design, SEO, Apps, UI/UX).
  • Prova Social: Quem são os membros? Quais ferramentas dominam?

3 Caminhos Possíveis

A Agência Fictícia

Um site corporativo moderno para vender serviços de tecnologia.

  • ✓ Cards de Serviços com Ícones
  • ✓ Seção "Sobre Nós" com fotos da equipe
  • ✓ Formulário de Orçamento estilizado

Coletivo Criativo

Foco nos talentos individuais do grupo (Portfólio conjunto).

  • ✓ Galeria de Projetos (Grid Masonry)
  • ✓ Links para Github/Linkedin de cada um
  • ✓ Estilo artístico e ousado

Startup Studio

Fingir que o grupo cria produtos/apps próprios.

  • ✓ Showcase de "Apps Fictícios"
  • ✓ Design minimalista (estilo Apple/SaaS)
  • ✓ Animações de scroll suaves

Como executar?

1

Branding (Design)

A primeira semana é sobre identidade. Criem um nome para a agência. Definam uma paleta de cores (evitem o "padrão"). Escolham fontes modernas (Sans-serif geométricas). Criem um logotipo simples no Figma.

2

O Poder do Grid (Autoria Web)

Portfólios precisam de organização. Use display: grid para criar uma galeria de projetos responsiva. Use gap para o respiro. Teste layouts assimétricos (ex: uma imagem grande e duas pequenas ao lado) para dar dinamismo.

3

Micro-interações (JS + CSS)

Adicione personalidade. Faça os cards "subirem" levemente no hover. Crie um menu de navegação que muda de cor ao rolar a página. Se possível, adicione um tema claro/escuro (Dark Mode Toggle) com JS.

Dica de Ouro:

Use "Placeholders" inteligentes. Se vocês não têm projetos reais, criem capas de projetos fictícios no Figma (ex: "Redesign do Instagram", "App de Delivery"). O importante é mostrar que vocês sabem construir a interface e apresentar o trabalho.
