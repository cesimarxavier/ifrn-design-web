Fundamentos de Layout:
Flexbox: O Container.

Adeus float. O Flexible Box Module (Flexbox) é um sistema unidimensional projetado para distribuir espaço e alinhar itens, mesmo quando seus tamanhos são desconhecidos ou dinâmicos.

1. A Física do Flexbox (Eixos)

O segredo para dominar Flexbox não é decorar propriedades, é entender os Eixos.

  • Main Axis (Eixo Principal) Onde os itens fluem. Definido por flex-direction.
  • Cross Axis (Eixo Transversal) Sempre perpendicular (90º) ao eixo principal.

justify-content

Controla o alinhamento no Eixo Principal (Horizontal se for row, Vertical se for column). É o responsável por distribuir os espaços vazios.

Principais Valores:

  • flex-start Agrupa no início.
  • center Centraliza o bloco.
  • space-between O 1º cola no início, o último no fim. (Essencial para Navbars).
3

align-items

Controla o comportamento no Eixo Transversal. Geralmente usado para centralizar verticalmente ou esticar itens para terem a mesma altura.

Atenção: Por padrão, é stretch (estica). Se você definir uma altura fixa no item, o stretch é ignorado.
