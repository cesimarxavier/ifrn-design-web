01. O Artigo de Notícia (Tipografia Escalável)

BÁSICO

Contexto: Recuperando a semântica da Semana 2, vamos estilizar um artigo usando unidades relativas para acessibilidade.

Tarefa: Configure o html para 16px. Crie um layout com título, data, subtítulo e parágrafos. Converta todos os tamanhos (fontes, margens, largura máxima) para rem.
  • Container centralizado com max-width: 40rem (aprox 640px).
  • Título (h1) com 2.5rem.
  • Texto base com 1.125rem (18px) para leitura confortável.

02. Botões e Menus (Display Inline-Block)

BÁSICO

Contexto: Links (<a>) são inline por padrão, o que impede definirmos padding vertical e largura correta.

Tarefa: Crie um menu de navegação simples. Transforme os links em botões usando display: inline-block.
  • Crie 4 links (Home, Sobre, Serviços, Contato).
  • Aplique padding interno, background e borda arredondada.
  • Use margin-right para separá-los.
  • Adicione efeito :hover para mudar a cor.
Meta Visual

03. Hero Section Imersiva (Backgrounds)

INTERMEDIÁRIO

Contexto: A "primeira dobra" de um site deve impactar. Vamos usar unidades de Viewport (vh) e manipulação de imagem.

Tarefa: Crie uma seção que ocupe 80% da altura da tela (80vh).
  • Use uma imagem de fundo de alta qualidade (Unsplash).
  • Aplique background-size: cover e position: center.
  • Adicione um overlay (gradiente preto semitransparente) sobre a imagem para garantir que o texto branco no centro seja legível.
Meta Visual

04. Tabela de Preços (Grid com Inline-Block)

INTERMEDIÁRIO

Contexto: Antes do Flexbox, usávamos inline-block para colocar elementos lado a lado. É importante entender essa "mecânica antiga" para lidar com alinhamentos.

Tarefa: Crie 3 cards de preços (Básico, Pro, Enterprise) alinhados horizontalmente.
  • Cada card deve ter width: 30% e display: inline-block.
  • Use box-sizing: border-box para evitar que o padding quebre o layout.
  • Destaque o card do meio ("Pro") com uma cor de borda ou sombra diferente.
Meta Visual

05. Card de Perfil (Position Relative/Absolute)

AVANÇADO

Contexto: Elementos flutuantes, como badges de notificação ou status "Online", dependem do relacionamento pai-filho no posicionamento.

Tarefa: Crie um Card de Perfil contendo foto e nome. Adicione um indicador "Online" (círculo verde) sobre a foto.
  • Container da foto deve ser position: relative.
  • O indicador (span vazio) deve ser position: absolute.
  • Posicione o indicador no canto inferior direito da foto usando bottom e right.
Meta Visual

06. Galeria com Legenda (Hover + Overlay)

AVANÇADO

Contexto: Efeitos interativos modernos frequentemente revelam informações ao passar o mouse.

Tarefa: Crie um container de imagem. Ao passar o mouse, uma tarja preta com texto deve deslizar ou aparecer na parte inferior da imagem.
  • Container: position: relative e overflow: hidden.
  • Legenda: position: absolute, fixada no bottom: 0.
  • Use a pseudo-classe :hover no container para alterar a opacidade ou posição da legenda.
Meta Visual

07. Header Fixo e Botão FAB

AVANÇADO

Contexto: Elementos que persistem na tela durante a rolagem são essenciais para UX (Navegação e Ações Rápidas).

Tarefa: Crie uma página longa (adicione muito texto). Adicione um Header que fica fixo no topo e um botão circular "+" flutuando no canto inferior direito.
  • Header: position: sticky ou fixed no topo. Use z-index alto.
  • Botão FAB (Floating Action Button): position: fixed, bottom: 2rem, right: 2rem.
  • Dê uma sombra forte (box-shadow) ao botão para destacá-lo.
Meta Visual

08. Desafio Final: O Modal (Overlay + Center)

DESAFIO

Contexto: Modais são janelas que bloqueiam a interação com o fundo. Exigem domínio total de posicionamento.

Tarefa: Construa a estrutura visual de um Modal.
  • Fundo: Uma div fixed ocupando 100vw e 100vh com fundo preto semitransparente.
  • Janela: Uma div branca centralizada exatamente no meio da tela (dica: use margens automáticas ou top: 50%; left: 50%; transform: translate(-50%, -50%);).
  • A janela deve ter sombra, bordas arredondadas e um botão de "Fechar".
