Tema 04

Produto Digital (SaaS):
Venda a solução, não o software.

O modelo de negócios mais forte da atualidade. Sua missão é criar a Landing Page de um aplicativo ou software (fictício) que transmita autoridade total.

O Cenário

Spotify, Netflix, Canva, Figma. Todos são produtos digitais (SaaS - Software as a Service). A página inicial desses produtos não é apenas "bonita"; ela é uma máquina de convencer pessoas.

Neste tema, o desafio é Arquitetura de Informação e Minimalismo. Como explicar algo complexo de forma simples? Como usar espaços em branco (whitespace) para guiar o olho do usuário?

A Sua Missão

Invente um software (ou recrie a página de um existente) e construa sua Landing Page. O site precisa ter uma estrutura rígida e profissional, contendo:

  • Proposta de Valor Clara: Um título (Hero) que explica o que o app faz em uma frase.
  • Grid de Recursos (Features): Ícones e textos curtos explicando as funcionalidades.
  • Tabela de Preços: O teste final de CSS (alinhamento, bordas e destaque).

3 Caminhos Possíveis

Aplicativo Mobile

Página para baixar um app de Fitness, Finanças ou Tarefas.

  • ✓ Botões "Download on App Store"
  • ✓ Mockup de Celular (Imagem do app)
  • ✓ Seção "Como funciona" (Passo a passo)

Software de Produtividade

Um concorrente para o Trello, Notion ou Slack.

  • ✓ Logos de empresas parceiras (Prova Social)
  • ✓ Tabela de Preços (Free vs Pro)
  • ✓ FAQ (Perguntas Frequentes)

Fintech / Banco Digital

Um cartão de crédito roxo ou um banco inovador.

  • ✓ Cartão com efeito de vidro (CSS Glass)
  • ✓ Números grandes animados
  • ✓ Segurança e Privacidade

Como executar?

1

Simplicidade Radical (Design)

Startups usam fontes limpas (Inter, Roboto, Poppins). Use muito espaço em branco entre as seções. O design deve ser "invisível" para destacar o conteúdo. Se usar ícones, que sejam todos do mesmo pacote (consistência).

2

Flexbox Master (Autoria Web)

Este é o melhor tema para dominar o Flexbox. Você vai usá-lo para alinhar o logo com o menu, para colocar o texto ao lado da imagem do celular, e para alinhar os preços na tabela.

3

Detalhes de UI (CSS)

Use border-radius para criar botões arredondados (estilo pílula). Aplique sombras suaves (box-shadow) nos cards de depoimentos para eles "flutuarem" sobre o fundo.

Dica de Ouro:

A "Dobra" (Above the Fold) é tudo. A primeira tela que o usuário vê (Título + Subtítulo + Botão + Imagem do Produto) deve ser perfeita. Gaste 50% do seu tempo de design só nessa parte.
