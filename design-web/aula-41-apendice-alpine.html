Guia Definitivo

De diretivas básicas a propriedades mágicas. O resumo técnico completo para suas aulas de Design Web.

1. O Básico (Core)

x-data Essencial

Cria o componente. Define o objeto de estado inicial.

<div x-data="{ 
  aberto: false, 
  user: 'Ana' 
}">...</div>
x-init Lifecycle

Roda código automaticamente ao carregar.

<div x-init="console.log('Nasci')">
  <!-- Útil para carregar APIs -->
</div>
x-bind (:)

Torna um atributo HTML dinâmico.

<button :disabled="isLoading"
        :class="{ 'bg-red': erro }">
  Enviar
</button>

2. Eventos e Modificadores

x-on (@)

Escuta eventos do DOM (click, submit, keyup).

<button @click="count++">+1</button>

<input @keyup.enter="add()">
Modificadores Poderosos Destaque
.prevent

Igual a e.preventDefault(). Essencial em forms.

.outside

Detecta clique fora do elemento. Perfeito para Modais.

.window

Escuta no navegador todo (ex: atalhos de teclado).

.debounce

Espera o usuário parar de digitar para rodar. Ótimo para busca.

3. Lógica e Loops

Mostrar/Esconder

x-show CSS (Display None) 
<div x-show="aberto">...</div>
x-if Remove do DOM 
<template x-if="aberto">
  <div>...</div>
</template>

Loops (Repetição)

Sempre use dentro de uma tag <template>. 

<ul>
  <template x-for="item in items">
    <li x-text="item.nome"></li>
  </template>
</ul>

4. Propriedades Mágicas ($)

$el

Pega o elemento HTML atual (igual this no JS).

$refs

Busca elementos marcados com x-ref.

<input x-ref="email">
<button @click="$refs.email.focus()">
$watch

"Vigia" uma variável. Se mudar, roda uma função.

x-init="$watch('busca', val => console.log(val))"
