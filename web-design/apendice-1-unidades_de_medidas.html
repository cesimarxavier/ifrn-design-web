Unidades de Medida CSS

Este guia serve como referência rápida para todas as unidades disponíveis na especificação do CSS3. Entender a "física" de cada unidade é essencial para ensinar e construir layouts robustos.

A

Relativas à Fonte (Tipografia)

Escalam baseadas no tamanho do texto. Essenciais para acessibilidade.

em Comum

Relativo ao tamanho da fonte do elemento pai.

Nota: Tem efeito "cascata". Se aninhado, multiplica o valor (ex: 1.2em * 1.2em).
RECOMENDADO
rem

Relativo ao tamanho da fonte do elemento raiz (<html>).

Uso: Padrão da indústria para layouts acessíveis. Ignora a cascata do pai.
ch Útil

Relativo à largura do caractere "0" (zero) da fonte atual.

Uso: Ótimo para limitar largura de texto (ex: max-width: 60ch) para leitura confortável.
ex

Relativo à altura da letra "x" minúscula da fonte atual.

Raramente usado, mas útil para ajustes finos de alinhamento vertical.
B

Relativas à Viewport (Tela)

Baseadas na área visível do navegador. Fundamentais para design responsivo.

Unidade Descrição Equivalência
vw Viewport Width. Largura da janela. 1vw = 1% da largura da tela
vh Viewport Height. Altura da janela. 1vh = 1% da altura da tela
vmin O menor valor entre vw e vh. Útil para garantir que algo caiba na tela sem scroll.
vmax O maior valor entre vw e vh. Útil para tipografia gigante.
C

Unidades Absolutas

Têm tamanho fixo. Geralmente usadas para impressão, não para telas.

Aviso Pedagógico

Com exceção do px, evite usar estas unidades para Web Design responsivo. Elas não se adaptam às configurações de DPI ou tamanho de tela do usuário. Use-as apenas em folhas de estilo para impressão (@media print).

px

Pixels
(Tela)

cm

Centímetros

mm

in

Polegadas
(1in = 96px)

pt

pc

Picas
(1pc = 12pt)

D

Outras Unidades (Ângulos e Tempo)

Usadas em transformações, animações e gradientes.

Ângulos (Rotate / Gradients)

  • deg Graus 0 a 360deg
  • rad Radianos 2π rad = 360deg
  • turn Voltas 1turn = 360deg

Tempo (Transition / Animation)

  • s Segundos Ex: 0.3s
  • ms Milissegundos 1000ms = 1s
